# app-model-01

This repository provides a ready-to-use full-stack template with a React frontend and a Spring Boot backend. It’s designed to be easily reusable for new applications without repeating the initial setup and configuration steps.

## Construcción de la app

### Backend

#### Tecnologías: Java - Spring - Spring Boot - PostgreSQL

Packages: 

1. Domain (Entidades, Interfaces y servicios)
	
	Contiene otros paquetes: 
		* Service

2. Web (Controladores)
	
	Contiene otros paquetes:
		* Controller

3. Persistence (Conexión con DB, APIs externas y correo)
